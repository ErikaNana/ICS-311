To build the application
------------------------
1. cd into the project directory
   cd Nana-Erika-A2/project
2. Create the bin directory
   mkdir bin
3. Compile all files 
   javac -d bin src/code/*.java
   javac -d bin -classpath bin src/code/Driver.java
   java -classpath bin code.Driver
